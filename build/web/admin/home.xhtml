WELCOME TO YOU E-VOTING SYSTEM PROJECT

Welcome to you online e-voting system. This project is made by me.This project is secured. Election commission control this project easily and automatically.He/She add the voter to the system. Voter can give vote by using id.A voter can give only one vote.If a voter try to give more than one vote the system will display error message. After completing vote election commission can declare the result automatically. In this project i have used JSF as framework. I have also used Hibernate to integrate with JSF. As back end i have used MySQL.This is my first project in JSF and Integrate with Hibernate.
